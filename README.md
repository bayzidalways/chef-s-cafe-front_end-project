## Getting Started

### Installation

1. **Clone the repository**

```bash
git clone git@github.com:bayzidalways/chef-s-cafe-front_end-project.git
cd chef's-cafe
````

2. **Install dependencies**

```bash
npm install
```

---

### Running the App

Start the development server:

```bash
npm run dev
```

Once compiled, visit `http://localhost:5173` in your browser.

---
