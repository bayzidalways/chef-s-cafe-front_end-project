## Getting Started

### Installation

Install dependencies:

```bash
npm install
```

### Development

Start the local development server:

```bash
npm run dev
```

Then open your browser at [http://localhost:5173](http://localhost:5173)

## Available Scripts

| Script            | Description                          |
| ----------------- | ------------------------------------ |
